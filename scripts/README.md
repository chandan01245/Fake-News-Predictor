# Scripts

Training and setup scripts for the ML model.

## Files

- `train_fast.py` - Fast training with stemming
- `train_ultra_fast.py` - Ultra-fast training without stemming
- `setup.bat` - Setup script for Windows
- `run.bat` - Run application script for Windows

## Usage

### Training
```bash
python scripts/train_ultra_fast.py
```

### Running the app
```bash
python src/web/app.py
```
